# JS字符串常用方法

1、 chatAt 从一个字符串中返回指定的字符

let str = 'banana'
str.charAt()    //"b"
str.charAt(1)   //"a"
str.charAt(2)   //"n"
str.charAt(10)  //""

2、concat 将一个或多个字符串与原字符串连接合并，形成一个新的字符串并返回

let hello = 'Hello, '
console.log(hello.concat('Kevin', '. Have a nice day.'))
// Hello, Kevin. Have a nice day.

let greetList = ['Hello', ' ', 'Venkat', '!']
"".concat(...greetList)  // "Hello Venkat!"

"".concat({})    // [object Object]
"".concat([])    // ""
"".concat(null)  // "null"
"".concat(true)  // "true"
"".concat(4, 5)  // "45"

3、includes 用于判断一个字符串是否包含在另一个字符串中，根据情况返回 true 或 false

let str = 'banana'
str.includes('apple')   //false
str.includes('bana')    //true
str.includes('bana',1)  //false

4、indexOf 返回调用它的 String 对象中第一次出现的指定值的索引，从 fromIndex 处进行搜索

let str = 'banana'
str.indexOf('na')   //2
str.indexOf('na',3) //4

5、lastIndexOf 返回调用 String 对象的指定值最后一次出现的索引，在一个字符串中的指定位置 fromIndex 处从后向前搜索。如果没找到这个特定值则返回-1

let str = 'banana'
str.lastIndexOf('na')   //4
str.lastIndexOf('na',3) //2

6、match 检索返回一个字符串匹配正则表达式的结果(数组)

let str = 'banana'
str.match(/a/)  //['a']
str.match(/a/)  //["a", "a", "a"]

7、replace 返回一个由替换值 replacement 替换部分或所有的模式 pattern 匹配项后的新字符串。模式可以是一个字符串或者一个正则表达式，替换值可以是一个字符串或者一个每次匹配都要调用的回调函数。如果 pattern 是字符串，则仅替换第一个匹配项

该方法并不改变调用它的字符串本身，而只是返回一个新的替换后的字符串

  • 第二个参数为字符串
let str = 'banana'
str.replace(/(b)an(a)na/g,`$2$1`)   //"ab"
  • 第二个参数为方法
function foo(match,p1,p2,offset,string){console.log(match,p1,p2,offset,string)}
let str = 'banana'
str.replace(/(b)an(a)n/,foo)    // banan b a 0 banana

8、search 执行正则表达式

let str = 'banana'
str.search()    //0
str.search('a') //1
str.search('x') //-1

9、split 使用指定的分隔符字符串将一个 String 对象分割成子字符串数组，以一个指定的分割字串来决定每个拆分的位置

let str = 'banana'
str.split() // ["banana"]
str.split('')   // ["b", "a", "n", "a", "n", "a"]
str.split('a')  // ["b", "n", "n", ""]
str.split('n')  // ["ba", "a", "a"]

10、slice 提取某个字符串的一部分，并返回一个新的字符串，且不会改动原字符串

let str = 'banana'
str.slice(1,2)  // 'a'
str.slice(-3)  // 'ana'

11、substring 返回一个字符串在开始索引到结束索引之间的一个子集, 或从开始索引直到字符串的末尾的一个子集

其中indexEnd 不支持负数，将负参数都直接转换为0

str.substring(1,3)  // 'an'
str.substring(-3)   // 'banana'

12、trim 删除一个字符串的两端空白字符，不修改原字符串

let str = ' banana '
str.trim()  // 'banana'

13、toLowerCase 调用该方法的字符串值转为小写形式，并返回。不修改原字符串

let str = "banana"
str.toUpperCase()  // 'BANANA'

14、toUpperCase 调用该方法的字符串转为大写形式并返回，不修改原字符串

let str = "Banana"
str.toLowerCase()  // 'banana'
